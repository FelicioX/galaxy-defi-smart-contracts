# Security Policy

## Reporting a Vulnerability

The security of this project is important.

If you discover a security vulnerability, please do not disclose it publicly before it has been reviewed and, if necessary, fixed.

Instead, report the issue privately with sufficient technical details to reproduce the problem.

## Contact

For security-related reports, please contact:

**Felicio Xavier**

GitHub: https://github.com/FelicioX

## Responsible Disclosure

Please allow a reasonable amount of time for the vulnerability to be investigated and resolved before making any public disclosure.

Responsible disclosure helps protect users and the integrity of the project.

Thank you for helping improve the security of Galaxy DeFi Smart Contracts.
